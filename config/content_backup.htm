IPv4 Ubuntu對外連線設定操作:

操作步驟:

  1. 喜好設定->網路->新增一個新的NAT網路
  2. 編輯NAT網路，新增4個連接埠轉送埠號
  3. 設定主機IP為0.0.1，主機連接埠號為 22、19999、17443、18443，客體IP為10.0.2.4，客體連接埠號為22、19999、17443、18443
  4. 點選設定>網路>設定為NAT Network
  5. 點選啟動，接著輸入密碼kmol2020
  6. 開啟LXTermial，輸入ifconfig> sudo apt install net-tools>密碼kmol2020
  7. cd tmp>ls>ping 127.0.0.1測試>cd cd2020pj1>git pull
  8. sudo vi wsgi.py編輯內容，將Host改成0.2.4
  9. 開啟leo，開啟cd cd2020pj1點NAV尋找allowExt
  10. 點選fileuploadfrom，在第10行新增一個ttt檔，存儲
  11. cd..，在tmp下新增oauth_scrum.txt並輸入sudo vi oauth_scrum.txt，儲存
  12. cd cd2020pj1>python3 wsgi.py
  13. 網址為https:/127.0.0.1:18443/alogin
  14. 帳號及密碼都為admin，進入後點選fileuploadform，點選要上傳的ttt
  15. 15.選download list檢查是否上傳完成

Ubnutu對外連線-IPV4

1.選擇主機網路NAT Network，新增埠號如下圖所示:

2.登入密碼一樣是kmol2020

3.打開LXTermial輸入ifconfig>sudo apt install net-tools輸入密碼也是kmol2020

4.輸入ping6 127.0.0.1測試是否可執行

5.進入cd2020pj1資料夾然後git pull

6.輸入sudo vi wsgi.py並且按下i編輯內容將’host=127.0.0.1’編輯設置為10.0.2.4

7.按下esc並且輸入:wq來存取，開啟cd2020pj1.leo

8.用nav尋找allowExt檔案位置並且新增ttt檔案

10.進入https://127.0.0.1:18443/alogin測試是否可以上傳ttt檔案

※ ifconfig為網卡信息的各項說

IPV6下伺服器連線設定

1.先設定好網路連線下的host ip( ip的設定方式可以在cd2020pj1倉儲下的downaloads中找到)

2.設定網路的ipv6位置設定 (ipv6 guest ip : fd17:625c:f037:2:a00:27ff:fef6:9b8a) (個人host ip : 2001:288:6004:17:2020:2::7221)格式只需更改後方:2::7221)

3.進入linux虛擬主機，依照指令輸入cd tmp>cd cd2020pj1>git pull >cd docs> cd cmsimde

4.輸入vi wsgi.py 並且更改host

5.設定好putty

6.按下save並且open後，在putty頁面分別輸入兩次密碼kmol2020

7.開啟Xming，接著輸入xterm&

8.在開啟的視窗中輸入cd tmp>cd 2020pj1>cd docs>cd cmsimde>python3 wsgi.py

用google輸入網址https://[2001:288:6004:17:2020:7221]:17443  (必須要用自己的ip)

9.一開始無法登陸，必須要關閉proxy，並且重新整理該頁面就可以了(若無法登陸可以嘗試關閉防火牆)
